# compare characterization view

Use for customizing UI

## Usage

``` r
compareCohortCharacterizationView(
  id,
  title = "Compare cohort characterization"
)
```

## Arguments

- id:

  Namespace Id - use namespaced id ns("compareCohortCharacterization")
  inside diagnosticsExplorer module

- title:

  Optional string title field
