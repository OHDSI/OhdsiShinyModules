# The module viewer for the shiny app report module

The module viewer for the shiny app report module

## Usage

``` r
reportViewer(id = "reportModule")
```

## Arguments

- id:

  the unique reference id for the module

## Value

The user interface to the home page module

## Details

The user specifies the id for the module
